## 🎯 为什么选择 BMad 方法？

与通用型 AI 编程助手不同，BMad Method 提供**结构化、实战验证的工作流，** 由精通敏捷开发的专属智能体驱动。每个智能体都具备深厚的领域专长——从产品管理到架构设计再到测试——它们协同运作，无缝衔接。

**✨** **核心优势：**

- **自适应智能** - 自动调整规划深度，从漏洞修复到企业系统皆可应对
- **完整开发生命周期** - 分析 → 规划 → 架构 → 实施
- **专业化专长** - 19 名具备特定职能的专员（项目经理、架构师、开发人员、用户体验设计师等）
- **成熟方法**论——基于敏捷最佳实践，并通过人工智能增强
- **IDE 集成** - 兼容 Claude Code、Cursor、Windsurf、VS Code

## 🏗️ BMad Core 的力量

**BMad Method** 实际上是构建在 **BMad Core**（ **协作 优化 反射引擎** ）之上的一个精密模块。这种革命性架构意味着：

- **BMad Core** 为人类与人工智能协作提供通用框架
- **BMad 方法**依托 Core 平台实现敏捷开发工作流
- **BMad Builder 让**您能够创建与 BMad Method 本身同样强大的自定义模块

借助 **BMad Builder**，您既能构建简单智能体，也能设计高度复杂的领域专用模块（法律、医疗、金融、教育、创意领域），这些模块即将通过**官方社区市场**实现共享。想象一下，您将能创建并分享专属的定制化 AI 团队！



## 🚀 三步快速入门

### 1. 安装 BMad 方法

```plain
# Install v6 Alpha (recommended)
npx bmad-method@alpha install

# Or stable v4 for production
npx bmad-method install
```

### 2. 初始化您的项目

在您的集成开发环境中加载任何代理并运行：

```plain
*workflow-init
```

此功能将分析您的项目，并推荐合适的工作流路径。

### 3. 选择你的轨道

BMad 方法通过三条智能轨道适应您的需求：

| **Track**  **轨道**                        | **Use For**  **用途**                      | **Planning**  **规划**                        | **Time to Start**  **开始之时** |
| ------------------------------------------ | ------------------------------------------ | --------------------------------------------- | ------------------------------- |
| **⚡** **Quick Flow**  **⚡** **快速流程**   | Bug fixes, small features 错误修复，小功能 | Tech spec only  仅技术规格                    | < 5 minutes  < 5 分钟           |
| **📋** **BMad Method**  **📋** **BMad 方法** | Products, platforms  产品，平台            | PRD + Architecture + UX PRD + 架构 + 用户体验 | < 15 minutes  < 15 分钟         |
| **🏢** **Enterprise**  **🏢** **企业**       | Compliance, scale  合规性，规模            | Full governance suite  完整治理套件           | < 30 minutes  < 30 分钟         |

**不确定？** 运行 `*workflow-init` 命令，让 BMad 分析您的项目目标。

## 🔄 运作原理：四阶段方法论

BMad 方法引导您遵循一个经过验证的开发生命周期：

1. **📊** **分析** （可选）——头脑风暴、调研并探索解决方案
2. **📝** **规划** - 创建产品需求文档、技术规范或游戏设计文档
3. **🏗️** **解决方案设计** ——架构设计、用户体验设计与技术方案设计
4. **⚡** **实施** - 以故事驱动开发，持续验证

每个阶段都有专门的工作流程和专员协同合作，以交付卓越成果。
