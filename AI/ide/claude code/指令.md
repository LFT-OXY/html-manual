# 直接创建 cc 命令

直接在终端输入 `cc` 就等同于执行 `claude --dangerously-skip-permissions`。

## 最简单的方法：使用 alias（推荐）

```bash
# 打开 zsh 配置文件
vim ~/.zshrc
```

**在文件末尾添加：**

```bash
alias cc='claude --dangerously-skip-permissions'
```

**保存并生效：**

```plain
# Ctrl + O 保存
# Ctrl + X 退出

# 立即生效
source ~/.zshrc
```



## 一键配置命令

如果您想更快，直接复制粘贴这一行命令：

```bash
echo "alias cc='claude --dangerously-skip-permissions'" >> ~/.zshrc && source ~/.zshrc && echo "✅ 完成！现在可以直接使用 cc 命令了"
```

## 验证

```plain
# 查看别名
alias cc

# 测试使用
cc your-prompt-here
```
