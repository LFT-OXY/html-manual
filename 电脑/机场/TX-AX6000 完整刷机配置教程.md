⏺ TX-AX6000 路由器科学上网配置教程

  一、准备工作

| 准备项       | 说明                                        |
| ------------ | ------------------------------------------- |
| 电脑         | Windows / Mac 均可                          |
| 网线         | 连接电脑和路由器（必须用网线，不能用 WiFi） |
| 浏览器       | Chrome 或 Edge                              |
| 机场订阅地址 | 提前复制好 Clash Subscription 地址          |

------

  二、下载官改固件

  2.1 打开下载页面

  访问：https://www.koolcenter.com/fw/device/tx-ax6000/asus_official

  2.2 选择固件版本

- 找到稳定版本（如 388_33411）
- 点击下载 .w 格式的固件文件
- 文件名类似：TX-AX6000_388_33411_koolcenter.w

  ⚠️ 注意：最新版 388_33965 有用户反馈间歇性断流问题，建议选择旧版本

  2.3 校验文件（可选但建议）

  下载页面会显示 MD5 值，用工具校验确保文件完整

------

  三、备份当前设置

  3.1 登录路由器后台

- 浏览器打开 http://192.168.50.1
- 输入用户名密码登录

  3.2 导出配置

- 进入：系统管理 → 还原/储存/上传设置
- 点击「储存设置」，下载配置备份文件

------

  四、刷入官改固件

  4.1 用网线连接

  电脑 ←——网线——→ 路由器 LAN 口（任意一个）

  4.2 进入固件升级页面

- 登录路由器后台 192.168.50.1
- 进入：系统管理 → 固件升级

  4.3 上传固件

- 点击「选择文件」
- 选择刚才下载的 .w 固件文件
- 点击「上传」

  4.4 等待刷机完成

- ⏳ 刷机过程约 3-5 分钟
- ⚠️ 期间不要断电、不要关闭浏览器、不要拔网线

  4.5 自动重启

- 刷机完成后路由器会自动重启
- 等待电源灯稳定（不再闪烁）
- 大约需要 2-3 分钟

------

  五、首次进入官改固件

  5.1 重新登录

- 浏览器打开 http://192.168.50.1
- 用原来的用户名密码登录
- 如果登录失败，试试默认账号：admin / admin

  5.2 确认刷机成功

- 左侧菜单应该出现「软件中心」选项
- 如果看到了，说明官改固件刷入成功 ✅

------

  六、安装科学上网插件

  6.1 插件选择

| 插件                 | 特点                 | 是否支持 Clash 订阅规则 |
| -------------------- | -------------------- | ----------------------- |
| MerlinClash / MCLite | Clash 内核，功能强大 | ✅ 完全支持              |
| vClash               | 轻量版 Clash         | ✅ 完全支持              |
| fancyss              | 传统插件，多协议支持 | ❌ 只提取节点            |

  💡 推荐安装 MerlinClash（MCLite），可以直接使用机场订阅里的分流规则

  6.2 安装 MerlinClash（推荐）

  步骤 1：获取安装包

  MerlinClash 需要从以下渠道下载：

- Telegram 群：搜索 MerlinClash 或 koolcenter
- GitHub：https://github.com/rts600/MerlinClash
- 下载适合 AX32 架构（TX-AX6000）的 .tar.gz 文件

  步骤 2：开启 JFFS

1. 路由器后台 → 系统管理 → 系统设置
2. 开启以下选项：
   \- ✅ Enable JFFS partition
   \- ✅ Enable JFFS custom scripts and configs
3. 点击「应用」，等待路由器重启

  步骤 3：离线安装

1. 进入软件中心
2. 点击「离线安装」标签
3. 上传下载的 .tar.gz 文件
4. 等待安装完成

  6.3 安装 fancyss（备选）

  方法 A：在线安装

- 在软件中心找到「科学上网」或「fancyss」
- 点击「安装」
- 等待安装完成

  方法 B：离线安装

1. 下载离线包：https://github.com/hq450/fancyss_history_package
   \- 找到 fancyss_mtk 文件夹
   \- 下载最新的 .tar.gz 文件
2. 软件中心 → 点击「离线安装」标签
3. 上传 .tar.gz 文件
4. 等待安装完成

------

  七、配置订阅

  7.1 获取订阅地址

- 登录机场官网
- 进入用户中心 / 仪表盘
- 找到「Clash Subscription」（推荐传统版，非 2022 版）
- 点击「复制订阅地址」

  订阅地址格式类似：
  https://sub.xxx.com/api/v1/client/subscribe?token=xxxxxxxx

  7.2 导入订阅

  MerlinClash 方式

1. 打开 MerlinClash 插件
2. 找到「配置文件」或「订阅管理」
3. 粘贴订阅地址
4. 点击「订阅」或「下载配置」

  fancyss 方式

1. 软件中心 → 点击「科学上网」
2. 找到「订阅设置」或「更新管理」
3. 粘贴订阅地址
4. 点击「保存并订阅」或「更新节点」

  7.3 等待节点导入

  导入成功后会显示节点列表，通常包括：香港、日本、新加坡、美国等节点

------

  八、启用科学上网

  8.1 选择节点

- 在节点列表中选择一个节点
- 推荐选择：香港或日本（延迟低）

  8.2 选择运行模式

| 模式       | 说明                     | 推荐   |
| ---------- | ------------------------ | ------ |
| 大陆白名单 | 国内网站直连，其他走代理 | ✅ 推荐 |
| GFWList    | 只有被墙的网站走代理     | 省流量 |
| 全局代理   | 所有流量都走代理         | 不推荐 |

  💡 如果使用 MerlinClash + Clash Subscription，订阅里已包含完整分流规则，无需额外设置模式

  8.3 开启插件

- 找到开关按钮，点击「开启」
- 等待状态变为「运行中」

------

  九、测试是否成功

  9.1 手机测试

- 手机连接路由器 WiFi
- 打开浏览器访问：https://www.google.com
- 能打开 = 成功 ✅

  9.2 电脑测试

- 访问：https://www.youtube.com
- 能打开并播放视频 = 成功 ✅

  9.3 检查 IP

- 访问：https://ip.sb
- 显示的 IP 应该是代理节点的地区（如香港）

------

  十、常见问题

| 问题               | 解决方法                                       |
| ------------------ | ---------------------------------------------- |
| 刷机后无法登录     | 长按路由器 Reset 键 10 秒恢复出厂              |
| 软件中心打不开     | 等待 1-2 分钟，刷新页面                        |
| 节点订阅失败       | 检查订阅地址是否完整，尝试换 Subscription 2022 |
| 能连但网速慢       | 换一个节点试试                                 |
| 手机能翻墙电脑不行 | 清除电脑 DNS 缓存：ipconfig /flushdns          |
| 国内网站变慢       | 模式改为「大陆白名单」                         |

------

  十一、日常使用

  11.1 自动翻墙设备

  配置完成后，所有连接路由器的设备自动翻墙：

- ✅ 手机（不用装任何 App）
- ✅ 电脑（不用装客户端）
- ✅ iPad / 平板
- ✅ 智能电视（可看 Netflix / YouTube）
- ✅ 游戏机（PS5 / Switch / Xbox）
- ✅ Apple TV / 小米盒子

  11.2 定期更新节点

- 每隔一段时间进入路由器后台
- 科学上网 → 点击「更新订阅」

  11.3 游戏加速器兼容

  如需使用 UU 加速器等游戏加速器，可在插件的「访问控制」中将游戏电脑设为绕过代理，避免冲突。
