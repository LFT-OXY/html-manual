## 1. 系统概述

### 1.1 系统介绍

地方政府专项债管理系统是一款专为地方政府专项债券项目全生命周期管理设计的企业级管理平台。系统涵盖项目申报、可行性研究、建设管理、收入预测、审核发布等核心业务流程。

### 1.2 主要功能

- **系统管理**: 用户、角色、权限、部门、菜单等基础管理
- **项目管理**: 专项债项目信息管理、科研项目管理、收入要素配置
- **建设管理**: 建设项目信息、建安要素管理
- **模板管理**: 收入模板、建安模板、支出模板等业务模板
- **AI辅助**: 自动生成可行性研究报告（支持豆包AI）
- **系统监控**: 操作日志、登录日志、在线用户、服务器状态监控

### 1.3 技术架构

- **前端**: Vue 2.6.12 + Element UI
- **后端**: Spring Boot 2.5.15 + MyBatis
- **数据库**: MySQL + Redis
- **认证**: JWT + Spring Security

### 1.4 浏览器要求

推荐使用以下浏览器访问系统：
- Google Chrome (最新版本)
- Microsoft Edge (最新版本)
- Firefox (最新版本)
- Safari (最新版本)

## 2. 快速开始

### 2.1 系统登录

#### 2.1.1 访问登录页

1. 打开浏览器，访问系统地址：`http://[服务器地址]:[端口]`
2. 系统默认运行在 `http://localhost:80`

#### 2.1.2 登录步骤

1. 在登录页面输入用户名和密码
2. 如果启用了Google双因素认证，需要输入6位验证码
3. 点击"登录"按钮
4. 登录成功后，系统自动跳转到首页

#### 2.1.3 初始账号

- **管理员账号**: admin
- **初始密码**: 请联系系统管理员获取

### 2.2 首页导航

登录成功后进入系统首页，页面主要包含：

- **顶部导航栏**: 系统标题、用户信息、退出登录
- **左侧菜单栏**: 功能模块导航（根据用户权限动态加载）
- **主内容区**: 显示当前选择的功能页面
- **仪表板**: 数据统计图表、待办事项、快捷入口

### 2.3 个人中心

点击顶部导航栏的用户名，可以访问个人中心：

- **基本信息**: 查看和修改个人资料
- **修改密码**: 修改登录密码
- **头像设置**: 上传个人头像
- **Google认证**: 绑定Google Authenticator（双因素认证）

## 3. 系统管理

系统管理模块提供了用户、角色、权限、部门等基础数据的管理功能。

### 3.1 用户管理

#### 3.1.1 查看用户列表

**路径**: 系统管理 > 用户管理

用户列表页面显示系统中所有用户的信息：
- 用户编号、用户名、昵称
- 所属部门
- 手机号码、邮箱
- 账号状态（正常/停用）
- 创建时间

**搜索功能**: 支持按用户名、手机号、部门、状态等条件筛选。

#### 3.1.2 新增用户

1. 点击"新增"按钮
2. 填写用户信息：
   - **必填项**: 用户名、昵称、密码、所属部门
   - **可选项**: 手机号、邮箱、性别、岗位、角色
3. 设置账号状态（正常/停用）
4. 点击"确定"保存

**注意事项**:
- 用户名不能重复
- 密码长度至少8位
- 新用户默认状态为"正常"

#### 3.1.3 编辑用户

1. 在用户列表中点击"修改"按钮
2. 修改用户信息
3. 点击"确定"保存更改

**提示**: 无法修改用户名，如需更改请删除后重新创建。

#### 3.1.4 删除用户

1. 勾选需要删除的用户（支持批量删除）
2. 点击"删除"按钮
3. 确认删除操作

**警告**: 删除用户后无法恢复，请谨慎操作。

#### 3.1.5 重置密码

1. 在用户列表中点击"重置密码"按钮
2. 输入新密码
3. 点击"确定"

**提示**: 密码重置后，建议通知用户及时修改。

#### 3.1.6 用户授权

为用户分配角色：
1. 点击用户的"更多" > "授权角色"
2. 勾选需要分配的角色
3. 点击"确定"

**说明**: 用户的权限由其拥有的角色决定，一个用户可以拥有多个角色。

#### 3.1.7 导入/导出用户

**导入用户**:
1. 点击"导入"按钮
2. 下载导入模板
3. 按照模板格式填写用户数据
4. 上传Excel文件
5. 系统自动导入用户

**导出用户**:
1. 勾选需要导出的用户（不勾选则导出全部）
2. 点击"导出"按钮
3. 系统生成Excel文件并下载

### 3.2 角色管理

#### 3.2.1 角色概述

角色是权限的集合，通过为用户分配角色来控制用户可以访问的功能模块。

**路径**: 系统管理 > 角色管理

#### 3.2.2 新增角色

1. 点击"新增"按钮
2. 填写角色信息：
   - **角色名称**: 角色的显示名称（如"项目经理"）
   - **权限字符**: 角色的唯一标识（如"project_manager"）
   - **角色排序**: 显示顺序（数字越小越靠前）
   - **状态**: 正常/停用
3. 配置菜单权限：勾选该角色可以访问的菜单和按钮
4. 设置数据权限：
   - **全部数据权限**: 查看所有数据
   - **自定义数据权限**: 查看指定部门的数据
   - **本部门数据权限**: 仅查看本部门数据
   - **本部门及以下数据权限**: 查看本部门及下级部门数据
   - **仅本人数据权限**: 仅查看自己创建的数据
5. 点击"确定"保存

#### 3.2.3 分配用户

为角色分配用户：
1. 点击角色的"更多" > "分配用户"
2. 查看已分配用户列表
3. 点击"添加用户"选择未分配的用户
4. 勾选用户后点击"确定"

**取消授权**: 在已分配用户列表中点击"取消授权"可以移除用户的该角色。

### 3.3 菜单管理

#### 3.3.1 菜单结构

系统菜单采用树形结构，支持多级菜单：
- **目录**: 菜单分组，不对应具体页面
- **菜单**: 对应具体页面
- **按钮**: 页面中的操作按钮（如新增、修改、删除）

**路径**: 系统管理 > 菜单管理

#### 3.3.2 新增菜单

1. 点击"新增"按钮（或在父级菜单上点击"新增"）
2. 选择菜单类型（目录/菜单/按钮）
3. 填写菜单信息：
   - **菜单名称**: 显示的菜单文本
   - **显示排序**: 菜单显示顺序
   - **权限字符**: 权限标识（如"system:user:list"）
   - **路由地址**: 前端路由路径
   - **组件路径**: 前端组件路径
   - **菜单图标**: 选择菜单图标
   - **是否外链**: 是否链接到外部URL
   - **是否缓存**: 是否缓存页面
   - **显示状态**: 显示/隐藏
4. 点击"确定"保存

### 3.4 部门管理

#### 3.4.1 部门树结构

部门采用树形结构组织，支持多层级部门。

**路径**: 系统管理 > 部门管理

#### 3.4.2 新增部门

1. 点击"新增"按钮（或在父级部门上点击"新增"）
2. 填写部门信息：
   - **上级部门**: 选择父级部门
   - **部门名称**: 部门名称
   - **显示排序**: 显示顺序
   - **负责人**: 部门负责人
   - **联系电话**: 部门电话
   - **邮箱**: 部门邮箱
   - **状态**: 正常/停用
3. 点击"确定"保存

### 3.5 岗位管理

岗位用于标识用户在组织中的职位。

**路径**: 系统管理 > 岗位管理

**操作**: 新增、修改、删除岗位，支持批量导出。

### 3.6 字典管理

#### 3.6.1 字典概述

数据字典用于管理系统中的下拉选项、状态码等枚举值。

**路径**: 系统管理 > 字典管理

#### 3.6.2 字典类型

字典分为类型和数据两层：
- **字典类型**: 字典的分类（如"项目状态"、"收入类型"）
- **字典数据**: 字典类型下的具体选项（如"进行中"、"已完成"）

#### 3.6.3 管理字典

1. 新增字典类型：点击"新增"，填写字典名称和类型
2. 配置字典数据：点击字典类型的"字典数据"，添加具体选项
3. 刷新缓存：修改字典后点击"刷新缓存"使更改生效

## 4. 项目管理

项目管理是系统的核心模块，用于管理地方政府专项债项目的全生命周期。

### 4.1 项目信息管理

#### 4.1.1 查看项目列表

**路径**: 项目管理 > 项目信息

项目列表显示所有专项债项目的基本信息：
- 项目名称、项目编号
- 所属部门、项目类型
- 投放领域、所属区域
- 项目金额、收入预测
- 项目状态、创建时间

**搜索功能**: 支持按项目名称、编号、类型、状态、时间范围等条件筛选。

#### 4.1.2 新增项目

1. 点击"新增"按钮
2. 选择项目版本类型（根据不同的专项债政策版本选择）：
   - 标准版
   - 专项债专用版（zxz）
   - 915版本
   - 98版本
   - 其他版本
3. 填写项目基本信息：
   - **项目名称**: 项目的完整名称
   - **项目编号**: 唯一标识码
   - **项目类型**: 从字典选择项目类型
   - **投放领域**: 选择专项债投放领域
   - **所属区域**: 选择行政区域
   - **建设期限**: 项目开始和结束时间
   - **项目总投资**: 项目总金额
   - **申请专项债额度**: 申请的债券额度
   - **项目描述**: 项目详细描述
4. 配置项目收入信息：
   - 添加收入来源
   - 选择收入要素
   - 填写收入预测数据（分年度）
5. 添加项目详情信息
6. 点击"确定"保存

**注意事项**:
- 项目编号不能重复
- 专项债额度不能超过项目总投资
- 收入预测应覆盖项目全周期

#### 4.1.3 编辑项目

1. 在项目列表中点击"修改"按钮
2. 修改项目信息
3. 点击"确定"保存更改

**提示**: 已审核通过的项目部分字段可能无法修改。

#### 4.1.4 复制项目

快速创建相似项目：
1. 点击项目的"更多" > "复制项目"
2. 系统自动复制项目信息（不包括审核状态）
3. 修改项目编号和名称
4. 点击"确定"保存

#### 4.1.5 项目审核

**审核流程**:
1. 项目创建后状态为"待审核"
2. 审核人员点击"审核"按钮
3. 填写审核意见
4. 选择审核结果：通过/驳回
5. 点击"确定"

**状态流转**:
- 待审核 → 审核通过 → 进行中 → 已完成
- 待审核 → 审核驳回 → 修改后重新提交

#### 4.1.6 批量操作

**批量修改状态**:
1. 勾选多个项目
2. 点击"批量修改状态"
3. 选择目标状态
4. 点击"确定"

**批量导出**:
1. 勾选需要导出的项目（不勾选则导出全部）
2. 点击"导出"按钮
3. 系统生成Excel文件

#### 4.1.7 生成项目文件

系统支持自动生成项目相关文档：
1. 点击项目的"生成文档"按钮
2. 选择文档类型：
   - 项目申报书
   - 可行性研究报告
   - 收入测算表
   - 其他材料
3. 系统自动生成Word/Excel文档
4. 下载文件

#### 4.1.8 查看覆盖倍数

覆盖倍数是衡量项目收入能否覆盖债券本息的重要指标：
1. 点击项目的"覆盖倍数"按钮
2. 查看各年度收入与还本付息的对比
3. 系统自动计算覆盖倍数（一般要求≥1.0）

#### 4.1.9 待办事项

查看项目相关的待办任务：
1. 点击"待办事项"按钮
2. 查看需要处理的任务列表
3. 点击任务可直接跳转到对应页面

### 4.2 科研/可研项目管理

#### 4.2.1 科研项目概述

科研项目（可行性研究项目）是专项债项目申报前的重要准备工作。

**路径**: 项目管理 > 科研项目

#### 4.2.2 新增科研项目

1. 点击"新增"按钮
2. 选择科研项目类型
3. 填写项目基本信息：
   - 项目名称
   - 研究周期
   - 研究内容
   - 预期成果
4. **AI自动生成**（可选）：
   - 点击"AI生成可研报告"
   - 系统调用豆包AI自动生成22个标准章节
   - 等待生成完成（支持实时进度显示）
   - 人工审核和修改生成内容
5. **手动编写**：
   - 依次填写可研报告的各个章节：
     1. 项目背景与必要性
     2. 需求分析与预测
     3. 建设规模与内容
     4. 技术方案
     5. 工程方案
     6. 投资估算
     7. 融资方案
     8. 财务分析
     9. 经济评价
     10. 社会效益分析
     11. 风险分析
     12. 研究结论与建议
     13-22. 其他专项内容
6. 点击"确定"保存

#### 4.2.3 AI生成功能说明

**功能特点**:
- 支持全部章节一键生成
- 支持单个章节独立生成/重新生成
- 实时显示生成进度
- 支持流式输出（边生成边显示）
- 生成内容可编辑修改

**使用建议**:
- AI生成的内容仅供参考，需要结合实际情况修改
- 重点关注数据、金额等关键信息的准确性
- 建议先生成初稿，再人工精细化调整

#### 4.2.4 指派编写人

1. 在科研项目列表中点击"指派编写人"
2. 选择编写人（可选择多人）
3. 点击"确定"
4. 系统自动发送通知给编写人

### 4.3 项目收入管理

#### 4.3.1 收入管理概述

项目收入是专项债偿还的资金来源，需要详细记录和预测。

**路径**: 项目管理 > 项目收入

#### 4.3.2 添加收入项

1. 点击"新增"按钮
2. 选择关联项目
3. 填写收入信息：
   - **收入类型**: 从字典选择（如经营性收入、政府性基金）
   - **收入来源**: 具体的收入来源描述
   - **收入要素**: 选择收入要素
   - **收入周期**: 收入开始和结束年份
   - **年度收入预测**: 填写各年度预计收入金额
4. 点击"确定"保存

### 4.4 收入要素管理

#### 4.4.1 收入要素概述

收入要素是收入测算的基础参数，用于标准化收入计算方法。

**路径**: 项目管理 > 收入要素

#### 4.4.2 管理收入要素

1. 新增收入要素：
   - 要素名称（如"门票收入"、"停车费收入"）
   - 计算公式
   - 参数定义
   - 适用范围
2. 编辑和删除要素
3. 在项目收入中引用要素

### 4.5 项目详情管理

**路径**: 项目管理 > 项目详情

项目详情用于记录项目的补充信息、进展情况、重要节点等。

## 6. 模板管理

模板管理提供了可复用的业务模板，提高项目创建效率。

### 6.1 模板体系结构

系统采用三级模板结构：
- **一级模板**: 顶级分类（如"交通基础设施"、"市政工程"）
- **二级模板**: 具体类型（如"道路工程"、"桥梁工程"）
- **三级模板**: 详细要素（具体的收入或建安要素组合）

### 6.2 一级模板管理

## 7. 系统监控

系统监控模块用于监控系统运行状态和用户操作行为。

### 7.1 操作日志

#### 7.1.1 查看操作日志

**路径**: 系统监控 > 操作日志

操作日志记录了用户的所有操作行为：
- 操作人员、操作模块
- 操作类型（新增/修改/删除/查询/导出）
- 操作IP、操作地点
- 请求参数、返回结果
- 操作时间、耗时

**搜索功能**: 支持按操作人、模块、类型、IP、时间范围等条件筛选。

#### 7.1.2 清理日志

1. 勾选需要删除的日志（支持批量删除）
2. 点击"删除"按钮
3. 确认删除操作

**建议**: 定期清理旧日志，避免数据库膨胀。

### 7.2 登录日志

#### 7.2.1 查看登录日志

**路径**: 系统监控 > 登录日志

登录日志记录了用户的登录行为：
- 登录账号
- 登录IP、登录地点
- 浏览器类型
- 操作系统
- 登录时间
- 登录状态（成功/失败）
- 失败原因

**作用**: 用于安全审计，发现异常登录行为。

### 7.3 在线用户

#### 7.3.1 查看在线用户

**路径**: 系统监控 > 在线用户

实时显示当前登录系统的用户：
- 会话编号
- 登录账号、用户名
- 所属部门
- 登录IP、登录地点
- 浏览器
- 登录时间

#### 7.3.2 强制退出

1. 找到需要退出的用户
2. 点击"强制退出"按钮
3. 确认操作

**用途**: 处理异常登录、释放会话资源。

### 7.4 定时任务

#### 7.4.1 任务管理

**路径**: 系统监控 > 定时任务

系统支持配置定时执行的任务，如：
- 数据统计
- 报表生成
- 数据清理
- 备份任务

#### 7.4.2 新增定时任务

1. 点击"新增"按钮
2. 填写任务信息：
   - **任务名称**: 任务的描述性名称
   - **任务组名**: 任务分组
   - **调用方法**: 要执行的方法名称
   - **Cron表达式**: 定时规则（如`0 0 2 * * ?`表示每天凌晨2点）
   - **是否并发**: 允许任务并发执行
   - **状态**: 启用/暂停
3. 点击"确定"保存

#### 7.4.3 执行任务

- **立即执行**: 点击"执行一次"按钮，立即执行任务
- **查看日志**: 点击"任务日志"查看执行历史

### 7.5 缓存监控

#### 7.5.1 查看缓存

**路径**: 系统监控 > 缓存监控

显示Redis缓存的使用情况：
- 缓存名称、缓存键
- 缓存值
- 过期时间

#### 7.5.2 清理缓存

1. 选择需要清理的缓存
2. 点击"清理"按钮
3. 确认操作

### 7.6 服务器监控

#### 7.6.1 查看服务器状态

**路径**: 系统监控 > 服务器监控

实时显示服务器运行状态：
- **CPU**: 使用率、核心数
- **内存**: 总内存、已用内存、使用率
- **磁盘**: 磁盘空间使用情况
- **Java**: JVM内存、线程数、类加载数
- **网络**: 网络接口、IP地址

**作用**: 监控系统资源使用，及时发现性能问题。

## 8. 常见问题

### 8.1 登录相关

**Q1: 忘记密码怎么办？**
A: 联系系统管理员重置密码，或使用"忘记密码"功能（如果已配置）。

**Q2: Google验证码错误怎么办？**
A:
1. 检查手机时间是否准确（需与服务器时间同步）
2. 确认使用的是正确的Google Authenticator账号
3. 重新扫描绑定二维码

**Q3: 登录后看不到菜单？**
A: 可能是未分配权限，联系管理员为您的账号分配角色和权限。

### 8.2 权限相关

**Q4: 为什么某些功能无法访问？**
A: 需要管理员为您分配相应的权限，联系管理员说明需要的功能模块。

**Q5: 如何查看自己有哪些权限？**
A: 在个人中心可以查看已分配的角色，但具体权限需要联系管理员确认。

### 8.3 项目管理相关

**Q6: 新增项目时如何选择版本？**
A: 根据您的专项债申报批次和政策要求选择对应版本：
- 标准版：适用于常规专项债项目
- zxz版：专项债专用版
- 915版、98版：特定批次的政策版本

**Q7: 项目审核被驳回后如何修改？**
A:
1. 在项目列表中找到被驳回的项目
2. 点击"修改"按钮
3. 根据审核意见修改项目信息
4. 保存后项目状态自动变为"待审核"
5. 等待重新审核

**Q8: 如何快速创建相似项目？**
A: 使用"复制项目"功能，系统会复制项目的所有信息（不包括审核状态），您只需修改项目编号和名称。

**Q9: AI生成的可研报告准确吗？**
A: AI生成的内容仅供参考，是基于通用模板和行业经验生成的初稿。您需要：
- 仔细核对所有数据和金额
- 根据项目实际情况修改
- 补充项目特有的内容
- 经过专业人员审核后才能正式使用

### 8.4 模板相关

**Q10: 如何选择合适的模板？**
A:
1. 根据项目类型选择一级模板（如"交通基础设施"）
2. 再选择对应的二级模板（如"道路工程"）
3. 如果没有完全匹配的模板，选择最接近的模板，然后手动调整

**Q11: 可以修改系统自带的模板吗？**
A: 建议不要直接修改系统模板，可以：
1. 复制系统模板
2. 修改复制后的模板
3. 保存为自定义模板使用

### 8.5 数据导入导出

**Q12: 导入数据失败怎么办？**
A: 检查以下几点：
1. 是否使用系统提供的模板
2. 数据格式是否正确（日期、数字等）
3. 必填字段是否填写完整
4. 是否有重复的唯一标识（如用户名、项目编号）
5. 查看错误提示信息，根据提示修改数据

**Q13: 导出的Excel文件打不开？**
A:
1. 确认Excel版本（建议使用Office 2016及以上版本）
2. 尝试使用WPS Office打开
3. 检查文件是否完整下载（文件大小是否正常）

### 8.6 系统性能

**Q14: 系统响应慢怎么办？**
A:
1. 检查网络连接
2. 清除浏览器缓存
3. 关闭不必要的浏览器标签页
4. 如果问题持续，联系系统管理员检查服务器状态

**Q15: 文件上传失败？**
A:
1. 检查文件大小（单个文件不超过10MB，总大小不超过20MB）
2. 检查文件格式是否支持
3. 检查网络连接是否稳定
4. 尝试更换浏览器

### 8.7 其他问题

**Q16: 如何联系技术支持？**
A:
- 在线支持：通过系统内的"帮助"菜单提交工单
- 电话支持：联系系统管理员获取支持热线
- 邮件支持：发送问题描述到技术支持邮箱

**Q17: 系统有移动端吗？**
A: 目前系统主要为PC端设计，移动端访问体验可能不佳。建议使用电脑浏览器访问。

**Q18: 如何提出功能需求或改进建议？**
A:
1. 通过系统内的"意见反馈"功能提交
2. 联系系统管理员汇总后提交给开发团队
3. 参与系统需求调研会议
