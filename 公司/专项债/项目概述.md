
基于数据库结构和项目文件分析，这是一个**政府专项债券项目管理系统**，主要用于管理地方政府专项债券项目的可行性研究报告（可研）和专项债申报文件的生成与管理。

## 核心业务模块

### 1️⃣ **项目基础管理**

**相关表：**
- `project_info` - 项目基本信息
- `project_detail` - 项目详细信息  
- `project_file` - 项目文件管理

**业务说明：**
项目从创建到最终审批的全流程管理，包括：
- 项目名称、编号、建设年限、运营年限
- 资金来源（资本金、专项债、市场融资）
- 项目类型和状态流转
- 项目审核流程（创建人→初审人→指定人→编写人→终审人）

### 2️⃣ **可行性研究报告（可研）生成**

**相关表：**
- `keyan` - 可研报告主表（含50+个`write_`字段）

**业务说明：**
自动生成可研报告的各个章节内容：
- **编制信息**：编制依据、法律依据、技术依据
- **项目必要性**：政策合理性、市场分析、建设必要性
- **项目建设方案**：建设规模、选址、技术方案、工程方案
- **财务分析**：投资估算、资金来源、收入预测
- **效益分析**：经济效益、社会效益、生态环境影响
- **结论与建议**：主要结论、风险对策

### 3️⃣ **工程造价管理（建安费）**

**相关表：**
- `construction_info` - 建安工程信息
- `construction_element` - 建安要素（工程项）
- `construction_zuofa` - 工程做法（材料清单）
- `construction_template` - 建安模板
- `construction_element_template` - 建安要素模板

**业务说明：**
管理工程建设和安装的费用明细：
- 分为**安装工程**和**设备采购**两大类
- 采用模板化管理，可复用常见工程配置
- 每个工程项包含：名称、数量、单位、单价
- 工程做法细化到具体材料和人工成本

### 4️⃣ **项目收入预测**

**相关表：**
- `project_revenue` - 项目收入信息
- `revenue_element` - 收入要素
- `revenue_template` - 收入模板
- `revenue_element_template` - 收入要素模板

**业务说明：**
预测项目运营期的收入情况：
- 计费周期设置（天/月/年）
- 税费计算（销项税、外购税、燃料动力税、附加税）
- 收入要素增长序列（模拟收入增长）
- 支持多种收入模式模板化配置

### 5️⃣ **财务测算与平衡分析**

**project_info表中的关键字段：**

```sql
-- 建设期投资分年投放
capital_fund1-6           -- 各年资本金投放
special_purpose_bonds1-6  -- 各年专项债投放
market_financing1-6       -- 各年市场融资

-- 运营期成本
wages                     -- 工资及福利
manage_money              -- 管理费
maintenance_costs         -- 维修保养费
raw_material_cost         -- 外购原材料费
fuel_cost                 -- 燃料动力费

-- 税费
value_added_tax           -- 增值税
additional_tax            -- 附加税
income_tax                -- 所得税

-- 融资参数
special_purpose_bonds_ir  -- 专项债利率
market_financing_ir       -- 市场融资利率
spb_distribution_rate     -- 专项债分账比例
```

**业务逻辑：**
- 计算项目全生命周期的现金流
- 验证专项债本息偿还能力
- 测算项目收益覆盖倍数
- 生成财务报表（资产负债表、利润表、现金流量表）

### 6️⃣ **投向领域管理**

**相关表：**
- `investment` - 投向领域（分层级结构）
- `level_one_template` - 一级模板（关联投向领域）

**业务说明：**
按照国家政策要求，专项债项目必须投向特定领域：
- 市政和产业园区基础设施
- 社会事业（教育、医疗、养老）
- 生态环保
- 农林水利
- 交通基础设施等

### 7️⃣ **文件生成与导出**

**相关表：**
- `file_template` - 文件模板（Word/Excel模板）

**业务说明：**
根据项目数据自动生成：
- 可行性研究报告（Word文档）
- 专项债申报材料
- 财务测算表（Excel）
- 绩效目标表

### 8️⃣ **支撑系统**

**相关表：**
- `sys_user` - 用户管理
- `sys_role` - 角色权限
- `sys_dept` - 部门组织架构
- `sys_area` - 行政区域
- `sys_dict_*` - 数据字典

## 核心业务流程

```
1. 项目创建
   ↓
2. 填写基本信息
   - 项目名称、建设内容
   - 业主单位、主管部门
   - 投向领域、行政区划
   ↓
3. 投资测算
   - 工程建安费（建设成本）
   - 工程其他费、预备费
   - 资金来源配置
   ↓
4. 收入预测
   - 配置收入模式
   - 设置收入要素和增长
   ↓
5. 成本配置
   - 人员工资
   - 运营管理费用
   - 维修费、原材料费
   ↓
6. AI生成内容（重要特色）
   - 调用火山引擎/豆包AI
   - 自动生成可研报告章节
   - 生成项目合理性论证
   ↓
7. 财务测算
   - 自动计算现金流
   - 验证债务偿还能力
   - 计算覆盖倍数
   ↓
8. 审核流程
   - 初审 → 指定编写人 → 编写 → 终审
   ↓
9. 文档导出
   - 生成可研报告Word
   - 生成专项债申报材料
```

## 项目特色功能

### 🤖 **AI集成**
从项目文件结构可见：
- `ruoyi-ui2/src/api/ai/doubao.js` - 豆包AI接口
- `ruoyi-ui2/src/api/ai/projectBond.js` - 专项债AI生成
- 自动生成可研报告各章节内容
- 智能论证项目必要性和可行性

### 📊 **模板化管理**
- 建安工程模板
- 收入模式模板  
- 文档模板
- 快速复用历史经验

### 💰 **精确财务测算**
- 分年度投资计划
- 全生命周期现金流
- 债务偿还能力分析
- 动态利率和增长序列

### 📝 **自动文档生成**
- 一键生成标准格式可研报告
- 自动填充项目数据
- 符合政府审批要求

## 使用场景

这个系统主要服务于：

1. **地方政府财政部门** - 专项债项目申报
2. **发改委** - 项目立项审批
3. **咨询公司** - 可研报告编制
4. **项目业主单位** - 项目前期策划

## 技术架构

- **后端**：Spring Boot（若依框架）
- **前端**：Vue 2 + Element UI  
- **数据库**：MySQL
- **AI服务**：火山引擎豆包大模型
- **文档生成**：POI (Word/Excel模板)
